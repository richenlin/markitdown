**描述**: Go metrics 语义精确性 — GetObjectStream 等流式 API 的 metrics 应区分两个阶段：1) 句柄获取成功（object_stream_opened）2) 数据读取完成（object_read_bytes），而非用单一计数器混淆（两阶段失败模式不同，混淆会导致告警规则无法精准定位是连接问题还是传输中断）
